## Important Notes

### Environment Variables
All required variables are in `.env`:
```
VITE_SUPABASE_URL=...
VITE_SUPABASE_PUBLISHABLE_KEY=...
SUPABASE_URL=...
SUPABASE_PUBLISHABLE_KEY=...
```

**These are PUBLIC keys - no secrets needed in .env**

### Database
- Uses Supabase (PostgreSQL managed)
- Row-level security enabled
- RLS policies protect data
- First account auto-becomes admin
